# Backup of files

It's a simple bash script to backup your files to the system

Here's an example of backing up all my pdf files tho the system 

# Install the following packages according to your system

```bash
find
notify-send
```

## Works on

- [x] Linux
- [x] macos
- [ ] Windows

# How to run
```
chmod +x backup.sh 
./backup.sh
```
